# 팁 기록
1. 타임시리즈 변수를 다양하게 표현할 수 있다.

2. requirement.txt를 github 계정에서 pip을 쓸 수 있을지.

3. Kaggle을 시작할 때, 조금 다양하게 사용해볼 수 있는 EDA 코드가 있으면 좋겠다.
3.1 Data 크기 확인해보기 : .shape()
3.2 Data : .info() or .head()
3.3 머신러닝 모델의 학습을 위해서는 훈련 데이터의 타입이 모두 int 또는 float이어야 한다.

4. 메모리 관리
4.1 info()를 중간에 삽입하여, 지속적으로 메모리량을 확인한다.
4.2 불필요한 변수명을 제거 또는 데이터의 타입을 적절하게 변경해준다.

5. 변수
5.1 수치형 변수일 경우 Describe()를 사용하여 기초 통계량을 확인해보며, 데이터의 분포를 파악한다.
- min, max, percentile을 확인하여, 데이터의 트랜드를 파악한다.

5.2 범주형 변수일 경우 Describe()를 사용
- 고유값, 빈도, 결측치 등을 통해 데이터의 분포를 파악한다.

결론 : 수치형 / 범주형 데이터의 분포와 트랜드 파악하기. 그리고 변수의 타입 바꿔 주기

코드 저장 : 범주형 데이터 타입에 들어있는 모든 변수들을 확인하는 코드
팁 : 데이터 분석 노트 양식으로 작성해보기

6. 시각화 (seaborn & matplotlib)
6.1 Jupyter Notebook에서 내부에 출력될 수 있도록 코드 1줄이 필요
-> %matplotlib inline

6.2 변수를 막대 그래프로 시각화
6.2.1 변수의 타입이 달라, 서로 다른 값으로 인지되는 경우도 존재.
6.2.2 눈에 띄는 작은 것이라도 꼼꼼히 기록하기(데이터의 Skew 정도, 빈도 분포, 트랜드)

7. 시계열 데이터 시각화
7.1 시간에 따른 변화 척도를 눈에 볼 수 있도록 분석
--> 방학/입학 시즌에 따라 데이터의 분포가 변할 수 있으나, 앞에 있었던 기초 통계 및 시각화에서는 시계열성 변화를 포착하지 못함.
